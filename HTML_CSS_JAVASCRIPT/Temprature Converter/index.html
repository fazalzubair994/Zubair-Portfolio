Temperature Converter

This project provides a simple yet powerful tool for converting temperatures between Celsius, Fahrenheit, and Kelvin. It features an intuitive user interface with three input fields, each representing one of the temperature scales. As users enter a value into any one of the input fields, the corresponding values in the other two fields are automatically calculated and updated in real-time.
